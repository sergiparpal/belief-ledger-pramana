# ADR 0004: Product positioning

Status: accepted, 2026-07-22.

The public headline is **“Evidence-backed policy enforcement for AI agents.”** Copy leads with
three operator outcomes: effectful actions are gated by current evidence and exact approval,
high-stakes output is grounded before accepted delivery, and later contradictions cause auditable
retraction and change subsequent decisions. The intended reader operates an agent that can affect
systems, people, money, or production state.

Plain language precedes the pramāṇa model. Sanskrit terms are reserved for the typed reasoning
implementation and may be introduced in a later “How it works” section. Hermes is described as the
first audited integration, not as the product boundary.

“Audit support” and “tamper-evident event history” are permitted. “Compliance layer” requires
external control evidence not supplied by this repository. The project must not claim to be a
sandbox, universal prompt-injection defense, exactly-once external-effects protocol, or durable
exactly-once response-delivery protocol. Enforcement claims must name the effective capability
profile defined by ADR 0006.
